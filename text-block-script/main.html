cursorBlock.js

Простая библиотека, чтобы показать блок при наведении и следовать за курсором

1. Что делает эта библиотека?

При наведении на элемент с атрибутом data-cursor-text, на экране появляется блок, который:

  • Появляется с анимацией
  • Следует за курсором
  • Меняет текст внутри себя на тот, что указан в data-cursor-text
  • Скрывается, когда курсор уходит

2. Подключение

Добавьте перед закрывающим тегом </body>:

<script src="https://danyaeast.github.io/scripts/text-block-script/textBlock.js"></script>

3. HTML-структура (обязательная)

<div class="section-one">
  <div data-cursor="object">
    <div data-cursor="replace">Этот текст будет заменён</div>
  </div>

  <div data-cursor-text="Привет">Наведи сюда</div>
  <div data-cursor-text="Пока">И сюда тоже</div>
</div>
  • data-cursor="object" — блок, который будет появляться и двигаться за курсором
  • data-cursor="replace" — текст в этом элементе заменяется при наведении
  • data-cursor-text — на этих элементах срабатывает эффект

4. Как запустить (JS)

Простой пример (всё включено по умолчанию)

const cursor = new cursorBlock('.section-one');

Пример с настройками

const cursor = new cursorBlock('.section-one', {
  appearEasing: "true",      // Появление с анимацией
  appearOpacity: "true",     // Изменение прозрачности
  appearSize: "true",        // Увеличение при появлении
  followEasing: "false",     // Мгновенное следование (без плавности)
  disappear: "false",        // Не исчезать при уходе
  position: "right bottom"    // Привязка угла блока к курсору
});

5. Все доступные настройки

ПараметрТипЧто делаетПо умолчанию
appearEasing"true" / "false"Плавное появление (с easing)"true"
appearOpacity"true" / "false"Меняет прозрачность от 0 до 100%"true"
appearSize"true" / "false"Увеличивается при появлении"true"
followEasing"true" / "false"Плавно догоняет курсор (иначе — резко)"true"
disappear"true" / "false"Скрывается при уходе с активной зоны"true"
positionstringГде блок будет относительно курсора"center"

Возможные значения position:
"left top", "right top", "left bottom", "right bottom",
"center", "left center", "right center", "center top", "center bottom"

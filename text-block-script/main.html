cursorBlock.js

Библиотека для создания кастомного блока, преследующего курсор и реагирующего на наведение

Подключение

<script src="https://danyaeast.github.io/scripts/text-block-script/textBlock.js"></script>

HTML-структура

<div class="section-one">
  <div data-cursor="object">
    <div data-cursor="replace">Текст будет заменён</div>
  </div>

  <div data-cursor-text="Привет">Наведи сюда</div>
  <div data-cursor-text="Пока">И сюда тоже</div>
</div>

Инициализация

const cursor = new cursorBlock('.section-one');

Все параметры по умолчанию уже включены.

Настройки

ПараметрТипОписаниеПо умолчанию
appearEasing"true" / "false"Плавное появление"true"
appearOpacity"true" / "false"Анимация непрозрачности"true"
appearSize"true" / "false"Анимация увеличения"true"
followEasing"true" / "false"Плавное следование за курсором"true"
disappear"true" / "false"Скрытие при уходе"true"
positionstringПоложение блока относительно курсора"center"

Возможные значения position: "left top", "right bottom", "center", и другие варианты (см. код).
